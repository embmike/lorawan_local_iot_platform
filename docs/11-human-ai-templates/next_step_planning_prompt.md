# Next-Step Planning Prompt

## Purpose

Use this prompt after a review or after a clarification cycle.

It determines the best next small step.

## Prompt

```text
Please recommend the next best small step for the project.

Consider:
- current gate status
- open points
- current risks
- current repository state
- current documentation state

Please provide:

1. Current situation summary
2. Best next small step
3. Why this is the best next step now
4. Preconditions
5. Expected output
6. Whether this step is a clarification step or an execution step
7. Recommended tool: ChatGPT or ChatGPT Codex

At the end, provide:
- a short action recommendation for the human
```
